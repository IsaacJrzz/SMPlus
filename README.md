# SM+ (Salud Madrid Plus) 🏥 — [TFG Edition]

**SM+** es una propuesta de rediseño vanguardista para el ecosistema digital de Salud Madrid. Este proyecto, desarrollado como **Trabajo de Fin de Grado (TFG)**, fusiona una estética "Dark Premium" con una arquitectura técnica de alto rendimiento para transformar la experiencia del ciudadano en la gestión de su salud mental y general.

## ✨ Características de Diseño (V2.0 - Actualizado)

El proyecto destaca por una interfaz disruptiva y profesional inspirada en las interfaces de usuario más avanzadas:

- **Glassmorphism & Hi-Tech UI:** Uso intensivo de efectos de cristal esmerilado (`backdrop-filter`) y tarjetas con iluminación interna que reaccionan al usuario.
- **Iconografía Profesional:** Sustitución de elementos informales por iconos vectoriales (SVG) de línea fina para un acabado corporativo y serio.
- **Navegación Móvil Ultra-Smooth:** Menú lateral minimalista de 3 barras con apertura al 100% de pantalla, eliminando fugas visuales y optimizando la navegación táctil.
- **Secciones Dinámicas:** Apartado de servicios con efectos de elevación y "glare" (brillo) para una experiencia de usuario más inmersiva.
- **Brand Integration:** Adaptación fiel de la paleta cromática de la Comunidad de Madrid: **Rojo Salud Madrid (#E30613)** y **Azul Sanidad (#00568F)**.

## 🛠️ Stack Tecnológico

- **React 18:** Arquitectura modular basada en componentes funcionales y hooks (`useState`).
- **Vite:** Entorno de construcción ultrarrápido para una experiencia de desarrollo fluida.
- **CSS3 Avanzado:** - **Flexbox & CSS Grid:** Para layouts complejos y adaptables.
  - **Animaciones Cubic-Bezier:** Transiciones de menú y tarjetas con aceleración natural.
  - **Responsive Design:** Media queries personalizadas para PC, Tablet y Móvil (ajuste total de cards y footers).

## 📂 Estructura de la Landing Page

- **Modern Navbar:** Botón toggle animado y desplegable de pantalla completa en color azul institucional.
- **Hero Section:** Impacto visual con mockups de dispositivos móviles integrados orgánicamente.
- **Services Grid:** Tres pilares fundamentales (Citas, Telemedicina e Historial) presentados en tarjetas tecnológicas.
- **Team & Contact:** Diseño adaptativo de tarjetas personales para los autores del proyecto, optimizado para evitar desbordamientos en móvil.
- **Modern Footer:** Estructura unificada en columna para dispositivos móviles con centrado absoluto de información institucional.

## 🚀 Guía de Inicio Rápido

1. **Clonar:** `git clone https://github.com/tu-usuario/sm-plus-landing.git`
2. **Instalar:** `npm install`
3. **Ejecutar:** `npm run dev`

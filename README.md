# SM+ (Salud Madrid Plus) 🏥

**SM+** es una propuesta de rediseño y mejora funcional para la aplicación oficial de Salud Madrid, desarrollada como parte de mi **Trabajo de Fin de Grado (TFG)**. El objetivo es optimizar la experiencia del ciudadano mediante una interfaz moderna, accesible y eficiente.

## 🎯 Objetivos del Proyecto
- **Accesibilidad:** Cumplimiento de estándares para usuarios de todas las edades.
- **Rendimiento:** Carga ultra rápida gracias al uso de Vite.
- **UX/UI:** Flujos de usuario simplificados para la gestión de citas y recetas.

## 🛠️ Stack Tecnológico
- **React 18:** Para la lógica de componentes.
- **Vite:** Como herramienta de construcción (Build Tool).
- **CSS3 / Modules:** Diseño responsive y mantenible.
- **Figma:** Utilizado para el prototipado previo (opcional, si lo usaste).

## 🚀 Guía de Inicio Rápido

1. **Clonar:** `git clone https://github.com/tu-usuario/sm-plus-landing.git`
2. **Instalar:** `npm install`
3. **Ejecutar:** `npm run dev`

## 📱 Vista Previa de Funcionalidades
- [ ] **Gestión de Citas:** Interfaz simplificada por especialidades.
- [ ] **Tarjeta Virtual:** Acceso rápido con biometría (simulado).

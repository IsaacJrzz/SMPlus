# SM+ (Salud Madrid Plus) 🏥 — [TFG Edition]

**SM+** es una propuesta de rediseño vanguardista para el ecosistema digital de Salud Madrid. Este proyecto, desarrollado como **Trabajo de Fin de Grado (TFG)**, fusiona una estética "Dark Premium" con una arquitectura técnica de alto rendimiento para transformar la experiencia del ciudadano en la gestión de su salud.

## ✨ Características de Diseño (V1.0)

El proyecto destaca por una interfaz disruptiva inspirada en tendencias modernas:

- **Glassmorphism UI:** Uso intensivo de efectos de cristal (`backdrop-filter`) y transparencias que aportan profundidad.
- **Lava Lamp Background:** Fondo dinámico con gradientes fluidos que representan la vitalidad y modernidad del sistema sanitario.
- **Diseño Responsive 360°:** Optimización total para PC, Tablet y Móvil, eliminando scrolls internos y garantizando fluidez táctil.
- **Brand Integration:** Adaptación fiel de la paleta cromática de la Comunidad de Madrid (Rojo Salud Madrid y Azul Sanidad).

## 🛠️ Stack Tecnológico

- **React 18:** Arquitectura basada en componentes funcionales.
- **Vite:** Entorno de desarrollo de última generación para tiempos de carga instantáneos.
- **CSS3 Moderno:** Implementación de Variables CSS, Flexbox, CSS Grid y animaciones de alto rendimiento.
- **Filtros Avanzados:** Uso de filtros de contraste e iluminación para integración orgánica de logos y mockups.

## 📂 Estructura del Proyecto (Landing Page)

- **Navbar:** Navegación por pestañas con estilo de "corte diagonal" (skews) y efectos de desenfoque.
- **Hero Section:** Espacio de impacto visual con integración de mockups interactivos y botones de acción (CTA).
- **Modern Footer:** Sección institucional adaptada al contexto del TFG con datos de autoría, tutoría y enlaces a la documentación de la memoria.

## 🚀 Guía de Inicio Rápido

1. **Clonar:** `git clone https://github.com/tu-usuario/sm-plus-landing.git`
2. **Instalar:** `npm install`
3. **Ejecutar:** `npm run dev`
